What is MAPI-Schmappy?

MAPI-Schmappy is command line tools to set Outlook address list search path and default address list.

The tools are explained in a blog post: http://explodingcoder.com/blog/content/programmatically-updating-outlooks-address-book-options-with-a-command-line-toolAll

June 23rd Release

The June 23rd binaries are build from http://github.com/christoffernielsen/MAPI-Schmappy/ containing the following changes:

  • Modified for Outlook 2010
  • Check that Microsoft Outlook is installed
  • Added solution and project files for Visual Studio 2013

MAPIAddrListSearch.exe sets the Outlook address list search list for name resolution when sending emails.

MAPIDefaultAddressList.exe sets the default address list when opening the address book.
